# Boston Housing Data Analysis

## Description

This project is a basic exploratory data analysis of the Boston Housing dataset using Python.

## Objectives

* Understand the dataset
* Analyze housing prices
* Explore relationships between variables such as crime rate, number of rooms, and price

## Tools Used

* Python
* pandas
* matplotlib

## Key Insights

* Housing prices tend to increase with the number of rooms
* Higher crime rates are generally associated with lower housing prices
* There are some exceptions, which suggests other factors also influence prices

## What I Learned

* How to load and explore real-world datasets using pandas
* How to compute basic statistics (mean, max, min)
* How to visualize data using matplotlib
* How to identify relationships between variables

## Dataset

The dataset used is the Boston Housing dataset, which contains information about housing in different areas of Boston.
